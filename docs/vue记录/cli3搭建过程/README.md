# cli3搭建过程

脚手架3：<br/>
创建：vue create 名称<br/>
运行：npm run serve<br/>

1，选择配置
```javascript
default (babel, eslint)  //默认配置
Manually select features  //手动选择
```
2，手动选择配置
```javascript
 (*) Babel
 ( ) TypeScript
 ( ) Progressive Web App (PWA) Support        
 ( ) Router
 ( ) Vuex
 ( ) CSS Pre-processors
 ( ) Linter / Formatter
 ( ) Unit Testing
 ( ) E2E Testing
 //按空格进行添加，回车确定
```

3，生成配置文件
```javascript
In dedicated config files  //单独的配置文件 喜欢用单独
In package.json   //配置在此文件
```

4，保存此配置
```javascript
 Save this as a preset for future projects?  
 //一般不保存
```
